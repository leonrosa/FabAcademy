    3. Computer-Aided Design¶

    The Assignment¶

     

    • Model (raster, vector, 2D, 3D, render, animate, simulate, ...) a possible final project.

     

    2D Softwares - Raster and Vector¶

     

    Krita¶

     

     

    Krita is a software for illustration creation, concept art, animation and it can also be used as photo editting tool. Krita works with raster images what is a image that uses bits to store information. With more bits, more resolution we get from the image but more disk space the image will take up.

    I chose Krita to create, to refine, and to render my concept skecth presented in my Principles and Practices page. Here it is the steps I used sketch my project:

    1. My first step was creating a sketch by hand.

       

       

    2. I took a picture of it and imported it into Krita.

       

       

    3. Krita has features very similar to Photoshop where I can control layers to work on the drawing. Then I included the picture in my second layer, to use it as reference for my next steps.

       

       

    4. The next step was to start drawing over the picture to refine the sketch. Then I created one more layer over it.

       

       

    5. To make drawings in Krita, I used a One by Wacom. With the tablet connected, I could access the Pop-up Palette by pressing the brush icon and clicking a button on the tablet's pen. With the palette displayed, I chose the pencil brush to start the drawings.

       

       

    6. Then I started drawing over my first draft and correcting the perspective and lines with a tool called Assitant Tool, a set of rulers that helps with straight lines, ellipses and perpective drawings.

       

       

    7. I took advantage from the layer feature to create some components and align them with other details that already had been drawn before.

       

       

    8. Finally, I repeated this process of overlaying the drawing on the draft with the pencil brush until I get the following result:

    Inkscape¶

     

     

    I used the Inkscape software to develop a logo for my documentation site. Inkscape is a software that use vectors to build illustrations and drawings. Vectors graphics are lines or shapes build by sequencial commands or mathematical statements. Those graphics have the advantage of keeping the resolution and quality no matter its size, performing as a great tool to develop a logo for the project.

    Here are the steps used for the logo development:

    1. First of all, I needed to use a image as reference to develop the logo. To do that I imported to Microsoft Paint, a native illustration software from Windows, the concept sketch developed in Krita and cutted the sideview using the selection tool.

       

       

      ![Paint Screenshot 2](.center .image}

       

       

    2. Then I draged this imaged into the Inkscape workspace.  

       

       

    3. I used a tool called Bezier to draw over the image.

       

       

       

    4. With the Bezier tool I created the shapes referring to each components.

       

       

    5. I assembled the shapes by dragging them and positioining them according to the reference image. The result looked like this:

       

       

    6. I created a frame around the vector drawing using the circle tool.

       

       

    7. With the circle geometry selected, I changed the stroke width by 5mm in the Fill and Stroke feature, inside the Stroke Style tab

       

       

    8. I centralized the circle with the drawing using the Center on Vertical Axis and Center on Horizontal Axis in the Align and Disttribute feature.

       

       

    9. I sent the circle back to the drawing and adjusted the size of the circle.

    3D Softwares¶

    Autodesk Fusion 360¶

     

     

    Autodesk Fusion 360 is the software I chose for modeling my plastic injector. Here it is the steps I followed to build my digital model:

    1. I brought the same image used in the Inkscape to Fusion by insertting a canvas into the workspace.

       

       

    2. Then the software asked me which plan I wanted to place the canvas. I placed it in the ZY plane.

       

       

    3. With the canvas placed, it is possible to model over it. Analogous process to the one used in the process with Krita.

       

       

    4. I started creating a sketch over the canvas.

       

       

    5. With the following tools (line, rectangle and circle) I created the profile, in ZX plane, according to the canvas dimensions. I also used the sketch dimension tool to define the measures of the geometry.

       

       

       

    6. I extruded the profile with the Extrude Tool with the right height dimension, following again the reference from canvas.

       

       

    7. Then I repeated the same process of sketching and extruding, now in the ZY plane.

       

       

       

    8. I repeat this two operation one more time. But the extrude operation was used as a cut tool, to create the side profife of the component.

       

       

       

    9. I used the Chamfer Tool to create a chamfer the corners of the two parts remaining after the cut.

       

       

       

    10. Then I used the Loft Tool to close the geometry by selecting the two profiles shaded in blue.

       

       

    11. I repeated this process to build the other components of my project. For the thread responsible to push the plastic into the chamber and the thread from the vise I needed to build a cylinder. Once again I drawed a circle as profile for extrusion.

       

       

       

    12. Then I created a thread with the Thread Tool by selecting the side surface from the body.

       

       

       

    13. The result of the 3D modeling using those tools (sketch, extrude, loft and thread) looked like this:

       

       

       

    Tinkercad¶

     

     

    In the Tinkercad, another 3D software from Autodesk which work in the browser and does not need to be installed, I modeled the PID controller enclosure. Here it is the steps:

    1. In the Fusion 360 I selected the upper part of the plastic injector with the right button of the mouse and exported it as a STL file.

       

       

       

    2. With Tinkercad opened, I imported the STL file by clicking over the button located at the upper right side of the interface. After that, the software asked me to select the file you want to import by dragging it over the specified area.

       

       

       

    3. Another window will appear where I set up the unities of measuments and the size of the model. In this case, I selected milimeters and kept the model size.

       

       

    4. With the model imported I adjusted the positioning the model to be able to work with it as I wanted.

       

       

    5. At the right side of the interface, I chose a cube and positioned it next to upper part of the plastic injector.

       

       

    6. Clicking over the cube, I dragged the blacked quare dots resizing the object according to the following dimensions.

       

       

       

       

       

    7. I created another cube and I repeated the process resizing the object according to the images below.

       

       

    8. I aligned both objects and moved the smaller prism 50mm along to z axis.

       

       

       

    9. I stretched the small prism to its body cross over the PID body. I displayed it as a Hole and I used the Group feature to cut the model.

       

       

    10. Then I created a wedge and repeated the last 3 steps to create a new cut on the enclosure body, to create a diagonal finishing at its upper side.

       

       

    11. Now I need to create a shel from this body. I duplicated the object by pressing Ctrl+D, displayed it as a hole, and resized it to be 3mm less in each side of the body.

       

       

    12. I created another cube as a hole, resized it and positioned it as a prolongation of the hole.

       

       

    13. Grouping those two hole objects with the enclosure, I got the following result.

       

       

    14. Following the same features and process presented before, I created a representation of the PID panel.

       

       

    15. Finally I exported a new STL mesh from the Tinkercad and imported again into Fusion, positioning the enclosure with the plastic injector body.
